## ⚙️ Destaque: Sistema de Requisições

Um dos focos deste portfólio é a apresentação do **Sistema de Requisições**, desenvolvido do zero para substituir solicitações informais e descentralizadas. 

**Resultados do fluxo implementado:**
- **100% de Rastreabilidade:** Histórico claro de cada etapa e cotação.
- **Aprovações Granulares:** Controle detalhado e aprovação por item específico.
- **Automação:** Geração automática de PDFs e notificações.
- **Eficiência:** Redução a zero do uso de planilhas intermediárias no fluxo.
